# SecureAgent Demo Guide

## Demo Objective
Show how SecureAgent detects, blocks, records, and reports unsafe AI-agent behavior.

## Step-by-Step Demo

### 1. Login
Login as admin.

### 2. Seed Demo Data
Use dashboard or testing page demo controls to seed demo data.

### 3. Open Dashboard
Show:
- total events
- blocked events
- allowed events
- high-risk events
- recent events
- active alerts
- live feed

### 4. Open Testing Lab
Run:
- benign example
- malicious example
- prompt injection
- secret exfiltration
- red-team suite

Explain how each run creates events and possibly incidents.

### 5. Open Incidents
Show:
- open incidents
- acknowledged incidents
- resolved incidents
- status actions

### 6. Open Reports
Show:
- severity breakdown
- role breakdown
- application breakdown
- export JSON/CSV

### 7. Open Policy Editor
Change a policy rule or threshold.
Reload the policy.
Run the same malicious prompt again.
Show how enforcement changes.

## Best Demo Story
SecureAgent is not just logging activity.
It is:
- monitoring
- blocking
- escalating
- reporting
- updating in real time
